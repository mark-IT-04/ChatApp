# ChatApp
Simple Chat Application

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

## Run Locally

Clone the project

```bash
  git clone https://github.com/mark-IT-04/ChatApp.git
```

Go to the project directory

```bash
  cd ChatApp
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the app

```bash
  npm run dev
```
